# Identidad de diseño propuesta

## Dirección visual

La identidad visual de EduPortal Frontend se orienta hacia una estética:

- moderna
- clara
- suave
- ligeramente neumórfica
- usable en contexto académico

No se busca un neumorfismo extremo. Se busca una **soft UI funcional**.

## Decisiones principales

### 1. Superficies suaves
Tarjetas y paneles con fondos claros semitransparentes y bordes sutiles.

### 2. Profundidad controlada
Sombras elevadas para contenedores principales y sombras internas para inputs o métricas.

### 3. Bordes redondeados amplios
Radios grandes para transmitir una sensación contemporánea y amable.

### 4. Paleta fría con acentos intensos
Base azul grisácea clara, acento principal índigo y estados de éxito/error con contraste suficiente.

### 5. Respiración visual
Espacios generosos y jerarquía limpia para que el layout se sienta más premium.

## Tokens visuales actuales

- Fondo base: `#e8eef8`
- Texto principal: `#1f2a44`
- Texto secundario: `#52607a`
- Acento primario: `#4f46e5`
- Error: `#dc2626`
- Éxito: `#0f766e`

## Motivo pedagógico

Esta identidad fue implementada primero en CSS para que el grupo vea:

- el costo de mantener estilos tradicionales,
- la cantidad de reglas que aparecen incluso en una sola vista,
- y la ventaja de traducir intención visual a utilidades en la siguiente fase.
